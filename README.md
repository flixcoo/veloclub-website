# Véloclub - Website

This repository contains the React website for "Véloclub", built with Vite and React. Its deployed version can be found at [veloclubband.de](https://veloclubband.de).

## Requirements
- Node.js (recommended: 18.x or newer LTS)
- npm

## Quick start

### Install dependencies
```bash
npm install
```


### Start development server
```bash
npm run dev
```

Default URL: `http://localhost:5173

### Create a production build
```bash
npm run build
```

Output directory: `/build`
